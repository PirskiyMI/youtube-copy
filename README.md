## Описание

Это учебный проект, который представляет собой упрощенную копию платформы YouTube. В приложении реализованы следующие функциональности:
- Вход и выход из аккаунта
- Подписка на каналы
- Лайки и дизлайки для видео
- Поиск видео
- Просмотр видео

## Технологии

- React
- React Router
- Redux Toolkit
- TypeScript
- HTML
- CSS
- SASS/SCSS
- Firebase
- Axios
- Vite
